1 Cum se downoadează un proiect de pe github

 Notă:  În "Pagina principală" -> 1.2 "Instalarea Serverului Spigot" învățăm cum să deschidem o consolă Git Bash
  1. în Git Bash navigăm navigăm în C:/Coderdojo/proiecte
  2. pe pagina github avem un buton verde cu un link
  3. dăm comanda: git clone <url>. <url> este luat de pa GitHub
