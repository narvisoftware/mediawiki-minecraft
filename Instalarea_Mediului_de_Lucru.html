1 Pregătirea mediului de lucru pentru atelierele CoderDojo: Java - Minecraft Modding cu Spigot

 Notă: Aceste instrucțiuni sunt făcute pentru sistemul de operare Windows.


1.1 Instalare Java Development Kit (Java JDK)

 Notă: 
  • Vom folosi java 8 (se mai numește 1.8). Atenție instalăm JDK, nu JRE.
  • Limk: http://www.oracle.com/technetwork/java/javase/downloads/index.html


1.2 Instalare NetBeans

Netbeans este editorul cu care creăm și modificăm fișiere java, gestionăm proiectul și compilăm fișierele sursă.

  • NetBeans se descarcă de aici: https://netbeans.org/downloads/
  • Instalăm versiunea Java EE.

 Notă: În timpul instalării: nu avem nevoie de serverele GlassFish si Tomcat (se debifează).
